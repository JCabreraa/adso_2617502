  • Git init: funciona para inicializar un proyecto en git.
  • Git config --global user.email "email": sirve para identificar y configurar globalmente el email con el repositorio virtual.
  • Git config --global user.name "name": se emplea para identificar y configurar globalmente el nombre del usuario con el repositorio virtual.
  • Git config user.email: verifica el email agregado.
  • Git config user.name: verifica el nombre de usuario agregado.
  • Git add .: se utiliza para agregar todos los archivos modificados y nuevos en el repositorio local.
  • Git status: permite verificar el estado del repositorio local.
  • Git commit -m "comentario" permite guardar los cambios realizados en el repositorio local con un mensaje resumido de los cambios.
  • git log:muestra el histrial de cambios en el repositorio local
  • git branchmuestra la rama del repisotorio local.
  • git chekout nombre_ramacambiar el nombre de la rama en un repositorio local.
  • git chekout -- .:deshacer los cambios en un archivo no confirmado
